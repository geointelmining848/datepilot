Support

If DatePilot is not working correctly, the fastest support path is email.

Email: geointelmining@gmail.com

What to include in a support request

  • Your device type and OS version
  • Whether the issue happened on iPhone or Android
  • What feature you were using
  • What you expected to happen
  • What actually happened
  • Any visible error message

What DatePilot supports

  • Profile rewrites and prompt improvements
  • Opening-line suggestions
  • Reply coaching
  • Ask-out timing and wording help
  • Subscription and restore-purchase issues

What DatePilot does not support

  • Logging into third-party dating apps for you
  • Automated messaging, swiping, or engagement
  • Scraping data from dating platforms

Troubleshooting basics

  • Check your internet connection if remote features are enabled.
  • Restart the app and try again.
  • Update to the latest app version if available.
  • If purchases look wrong, use the restore flow once it is available in release builds.
