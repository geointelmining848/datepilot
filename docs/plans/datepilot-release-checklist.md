# DatePilot Release Checklist

## Toolchain prerequisites

### iOS
- Full Xcode installed
- `xcode-select` points to Xcode.app
- Apple Developer account configured for signing

### Android
- Java runtime installed
- Android SDK / platform tools installed
- Play Console app signing requirements prepared

## Expo / app config
- App name finalized: DatePilot
- Slug confirmed
- Bundle identifiers / package names decided
- App icons and splash reviewed
- Production environment config decided
- AI mode / backend configuration documented

## Product readiness
- Core flows tested end-to-end
- Limits/paywall flow coherent
- Empty/loading/error states reviewed
- Copy consistency pass
- Privacy-sensitive wording reviewed

## Store readiness
- App Store description draft
- Google Play description draft
- keywords/category choices
- screenshots
- privacy policy
- support contact
- age rating questionnaire inputs

## Monetization readiness
- Premium scope finalized

- Payment wiring implemented

- iOS startup verified
- Android startup verified
- Crash-free basic navigation
- Persistence verified across app restarts
- Release build sanity check
